## Example curl request

```bash
curl --location --request POST 'https://cdn.growthbook.io/api/features/feat_a8ab78d93' \
--header 'Authorization: Bearer key_acce_asdfASDF1234!@#$ASDFasdf1234!@#$' \
--header 'Content-Type: application/json' \
--data-raw '{
    "owner": "johndoe@******.com",
    "valueType": "string",
    "defaultValue": "test",
}'
```
