# Update

Updates a feature with the given properties.

## Endpoint

`PUT /api/v1/features/{featureId}`

## Example curl request

```bash
curl --location --request PUT 'https://cdn.growthbook.io/api/v1/features/<featureId>' \
--header 'Authorization: Bearer <access_token>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "description": "New description",
    "owner": "New owner",
    "project": "new project name",
    "tags": ["new", "tag", "array"],
    "archived": true
}'
```
