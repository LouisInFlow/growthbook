# Get

Gets a feature by its ID.

## Endpoint

`GET /api/v1/features/{featureId}`

## Get feature request

<section className="text-sm">
  <div className="text-lg font-bold">Headers</div>
  <ParamsRow
    name="Authorization"
    type="Bearer <access_token>"
    description="access_token for API authentication"
    hr={false}
  />
</section>

## Example curl request

```bash
curl --location --request GET 'https://cdn.growthbook.io/api/v1/features/<featureId>' \
--header 'Authorization: Bearer <access_token>' \
--header 'Content-Type: application/json'
```
